MYSQL RULE BOOK
===============

Rule Book Name:
MySQL Data Quality Rule Book

Connector Type:
mysql

Dataset Type:
table

Description:
Built-in quality and governance rules derived from monitoring_controller.py.

RULES
-----

1. Row Count Validation
- Rule Type: custom_sql
- Purpose: Detect empty tables.
- SQL:
SELECT 1 WHERE (SELECT COUNT(*) FROM {{schema}}.{{table}}) = 0
- Penalty in controller: -20

2. Null Value Check
- Rule Type: null_check
- Parameters:
{
  "column": "<column_name>",
  "max_nulls": 0
}
- Controller also flags columns with >30% NULL values.
- Penalty in controller: -5

3. Duplicate Record Check
- Rule Type: unique_check
- Parameters:
{
  "column": "<column_name>"
}
- Penalty in controller: -3

4. Numeric Range Validation
- Rule Type: range_check
- Parameters:
{
  "column": "<column_name>",
  "min": 0,
  "max": 1000000
}

5. Invalid Date Check
- Rule Type: custom_sql
- SQL:
SELECT * FROM {{schema}}.{{table}}
WHERE {{date_column}} < '1900-01-01'
   OR {{date_column}} > '2100-12-31'
LIMIT 1
- Penalty in controller: -4

6. Primary Key Uniqueness
- Rule Type: custom_sql
- Purpose: Detect duplicate primary keys.
- Penalty in controller: -10

7. Foreign Key Integrity
- Rule Type: custom_sql
- Purpose: Detect orphan records.
- Penalty in controller: -8

8. Truncation Check
- Rule Type: custom_sql
- Purpose: Detect values equal to column max length.
- Penalty in controller: -3

9. Data Freshness Check
- Rule Type: custom_sql
- Purpose: Detect if latest timestamp is older than 7 days.
- Penalty in controller: -5

10. Outlier Detection
- Rule Type: custom_sql
- Purpose: Detect abnormal numeric values using IQR.
- Penalty in controller: -3

11. Regex Validation
- Rule Type: regex_check
- Parameters:
{
  "column": "<column_name>",
  "pattern": "^[A-Z]{5}[0-9]{4}[A-Z]$"
}

12. Custom SQL Rule
- Rule Type: custom_sql
- Purpose: Any business-specific validation.
- Additional Penalty if failed: -10

PII DETECTION
-------------
Automatic PII detection scans:
- Column names (email, phone, aadhaar, pan, passport, dob, address)
- Sample values (up to 30 rows)

Quality Score Formula:
100 - sum(all penalties), bounded between 0 and 100.
